== Description ==

EN:

Adds a simple widget to the sidebar that displays a random posts.

RU:

Добавляет простой виджет в боковую панель сайта, который отображает случайные посты.

== Installation ==

EN:

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Done!

RU:

1. В админ-центре необходимо загрузить и активировать данный плагин.
2. Настройка виджета осуществляется в меню: «Внешний вид» — «Виджеты».
3. Готово.
